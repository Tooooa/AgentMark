# AgentMark 项目启动指南

本文档介绍如何启动 AgentMark 项目的后端服务和前端界面。

## 1. 环境准备

请确保您的系统已安装以下环境：

- **Python**: 3.9 或更高版本
- **Node.js**: 18.0 或更高版本 (推荐使用 LTS)
- **NPM**: 通常随 Node.js 一起安装

### 安装 Python 依赖

项目根目录 `/root/autodl-tmp/AgentMark2/AgentMark` 下有 `requirements.txt`，但 Dashboard 后端需要一些额外的 Web 服务依赖。请运行以下命令安装所需库：

```bash
# 进入项目根目录
cd /root/autodl-tmp/AgentMark2/AgentMark

# 安装基础依赖
pip install -r requirements.txt

# 安装 Dashboard 后端专用依赖
pip install fastapi uvicorn sentence-transformers pydantic openai python-multipart
```

## 2. 启动后端 (Backend)

后端服务基于 FastAPI，处理模型调用和数据检索。

1.  **打开一个新的终端窗口**。
2.  进入项目根目录：
    ```bash
    cd /root/autodl-tmp/AgentMark2/AgentMark
    ```
3.  启动服务：
    ```bash
    python dashboard/server/app.py
    ```

**成功提示**：
当您看到类似 `Uvicorn running on http://0.0.0.0:8000` 的输出时，说明后端已成功启动。

> **注意**：后端服务默认监听 **8000** 端口。

## 3. 启动前端 (Frontend)

前端界面基于 React + Vite。

1.  **打开一个新的终端窗口**。
2.  进入 Dashboard 目录：
    ```bash
    cd /root/autodl-tmp/AgentMark2/AgentMark/dashboard
    ```
3.  安装前端依赖 (仅首次需要)：
    ```bash
    npm install
    ```
4.  启动开发服务器：
    ```bash
    npm run dev
    ```

**成功提示**：
终端会显示访问地址，通常为：
```
  ➜  Local:   http://localhost:5173/
```

## 4. 访问应用

打开浏览器，访问前端终端中显示的地址 (例如 `http://localhost:5173` 或 `http://127.0.0.1:5173`) 即可使用 AgentMark 界面。

## 常见问题

- **端口被占用**：如果 8000 或 5173 端口被占用，请检查是否有其他服务正在运行，或者修改配置文件 (前端在 `vite.config.ts`, 后端在 `app.py` 底部)。
- **依赖缺失**：如果启动后端时报错 `ModuleNotFoundError`，请根据错误提示使用 `pip install <缺少包名>` 进行安装。
